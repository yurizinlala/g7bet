Artigos, livros e trabalhos publicados pelo professor

Um Framework para Escalonamento de Tarefas em Sistemas Embarcados de Tempo Real

Artigo

Autores: Felipe Dennis M. de Oliveira, João Silva, Maria Santos

Revista: Journal of Real-Time Systems

Ano: 2024

DOI: 10.1000/xyz123

Este artigo apresenta um framework para escalonamento de tarefas em sistemas embarcados com restrições de tempo real, focando em aplicações com recursos limitados. O framework proposto utiliza uma abordagem híbrida que combina técnicas de escalonamento estático e dinâmico, adaptando-se às condições de carga do sistema.

Sistemas Embarcados: Teoria e Prática

Livro

Autores: Felipe Dennis M. de Oliveira, Carlos Mendes

Editora: Editora Universitária

Ano: 2023

ISBN: 978-85-7777-777-7

Este livro apresenta os fundamentos teóricos e práticos dos sistemas embarcados, abordando desde conceitos básicos até aplicações avançadas. A obra inclui exemplos práticos, estudos de caso e exercícios, sendo indicada para estudantes e profissionais da área de computação e engenharia.

Implementação de um Sistema IoT para Monitoramento Ambiental em Áreas Rurais

Evento

Autores: Felipe Dennis M. de Oliveira, Ana Costa, Pedro Almeida

Conferência: Simpósio Brasileiro de Sistemas Embarcados (SBSE)

Local: São Paulo, Brasil

Ano: 2023

Este trabalho apresenta o desenvolvimento e implementação de um sistema IoT de baixo custo para monitoramento ambiental em áreas rurais. O sistema utiliza sensores conectados a uma rede sem fio para coletar dados sobre temperatura, umidade, luminosidade e qualidade do ar, enviando-os para uma plataforma web para análise e visualização.

Análise de Desempenho de Algoritmos de Escalonamento em Sistemas de Tempo Real

Artigo

Autores: Felipe Dennis M. de Oliveira, Roberto Fernandes

Revista: Revista Brasileira de Computação Aplicada

Ano: 2022

DOI: 10.1000/abc456

Este artigo apresenta uma análise comparativa de desempenho entre diferentes algoritmos de escalonamento utilizados em sistemas de tempo real. São avaliados aspectos como previsibilidade temporal, utilização de recursos e comportamento sob condições de sobrecarga, utilizando um ambiente de simulação desenvolvido especificamente para este estudo.

Desenvolvimento de um Laboratório Virtual para Ensino de Sistemas Embarcados

Evento

Autores: Felipe Dennis M. de Oliveira, Mariana Lima, José Santos

Conferência: Congresso Brasileiro de Educação em Engenharia (COBENGE)

Local: Brasília, Brasil

Ano: 2022

Este trabalho apresenta o desenvolvimento e implementação de um laboratório virtual para o ensino de sistemas embarcados, permitindo que estudantes realizem experimentos práticos remotamente. São descritos os aspectos técnicos da solução, bem como os resultados pedagógicos obtidos durante sua aplicação em disciplinas de graduação.

Sistema de Monitoramento Remoto para Aplicações IoT

Pesquisa Em andamento

Período: 2024 - Atual

Financiamento: UERN

Este projeto visa desenvolver uma plataforma de monitoramento remoto para aplicações de Internet das Coisas (IoT), permitindo a coleta, processamento e visualização de dados de sensores distribuídos. A solução utiliza tecnologias embarcadas de baixo custo e consumo energético, integradas a uma infraestrutura de nuvem para armazenamento e análise de dados.

Objetivos

  • Desenvolver hardware de sensoriamento com comunicação sem fio
  • Implementar gateway para coleta e transmissão de dados
  • Criar plataforma web para visualização e análise
  • Validar o sistema em aplicações reais

Equipe

Prof. Felipe Dennis (Coordenador), 2 alunos de graduação, 1 aluno de mestrado

Laboratório Virtual de Sistemas Embarcados

Ensino Concluído

Período: 2022 - 2023

Financiamento: UERN

Desenvolvimento de um ambiente virtual para ensino de sistemas embarcados, permitindo que estudantes realizem experimentos práticos remotamente. O projeto incluiu a criação de simuladores, tutoriais interativos e hardware acessível remotamente para programação e testes.

Resultados

  • Plataforma web com 15 experimentos interativos
  • Infraestrutura para acesso remoto a hardware real
  • Material didático completo para disciplinas relacionadas
  • Utilizado por mais de 200 alunos durante o período de pandemia

Equipe

Prof. Felipe Dennis (Coordenador), 3 alunos de graduação

Automação para Pequenos Produtores Rurais

Extensão Em andamento

Período: 2023 - Atual

Financiamento: UERN / Parceria com Cooperativa Local

Projeto de extensão que visa desenvolver e implementar soluções de automação de baixo custo para pequenos produtores rurais da região. As soluções incluem sistemas de irrigação automatizada, monitoramento de condições ambientais e controle de estufas, utilizando tecnologias acessíveis e de fácil manutenção.

Ações

  • Oficinas de capacitação para produtores rurais
  • Desenvolvimento de protótipos adaptados à realidade local
  • Implementação de sistemas-piloto em propriedades selecionadas
  • Acompanhamento e suporte técnico contínuo

Equipe

Prof. Felipe Dennis (Coordenador), 4 alunos de graduação, 1 técnico

Escalonamento de Tarefas em Sistemas de Tempo Real

Pesquisa Concluído

Período: 2021 - 2022

Financiamento: CNPq

Pesquisa focada no desenvolvimento e análise de algoritmos de escalonamento para sistemas de tempo real, com ênfase em aplicações críticas. O projeto investigou técnicas para garantir previsibilidade temporal em sistemas com recursos limitados e restrições de energia.

Resultados

  • Desenvolvimento de novo algoritmo de escalonamento
  • Implementação de framework para testes e validação
  • Publicação de 2 artigos em conferências internacionais
  • Geração de software open-source para a comunidade

Equipe

Prof. Felipe Dennis (Coordenador), 1 aluno de mestrado, 2 alunos de graduação
